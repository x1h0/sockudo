Some Sockudo client SDKs are newly released and may not yet be available on all package managers. Until they are published, install directly from GitHub — links are provided for each SDK below.
::

## JavaScript / TypeScript

```bash
npm install @sockudo/client
```

Imports:

```ts
// Default (browser)
import { SockudoClient } from "@sockudo/client";

// Filter utilities
import { Filter } from "@sockudo/client/filter";

// Encrypted channels
import { SockudoClientEncrypted } from "@sockudo/client/with-encryption";

// Worker
import SockudoWorker from "@sockudo/client/worker";

// React hooks
import { SockudoProvider, useChannel } from "@sockudo/client/react";

// Vue composables
import { createSockudoPlugin, useChannel as useSockudoChannel } from "@sockudo/client/vue";
```

Framework peers:

```bash
npm install react react-dom vue
```

## Swift

Uses Swift Package Manager directly from GitHub — no additional registry needed.

[sockudo/sockudo-swift](https://github.com/sockudo/sockudo-swift)

```swift
.package(url: "https://github.com/sockudo/sockudo-swift.git", from: "0.1.0")
```

```swift
import SockudoSwift
```

## Kotlin

[sockudo/sockudo-kotlin](https://github.com/sockudo/sockudo-kotlin) — pending Maven Central publish.

```kotlin
// Once published to Maven Central:
implementation("io.sockudo:sockudo-kotlin:0.1.0")
```

Until then, install from GitHub Packages or build locally:

```bash
git clone https://github.com/sockudo/sockudo-kotlin
cd sockudo-kotlin && ./gradlew :lib:publishToMavenLocal
```

## Basic Connect

::code-group

```js [JavaScript]
import SockudoClient from "@sockudo/client";

const client = new SockudoClient("app-key", {
  wsHost: "127.0.0.1",
  wsPort: 6001,
  forceTLS: false,
  // protocolVersion: 2 is the default
});

client.connect();
```

```swift [Swift]
let client = try SockudoClient(
    "app-key",
    options: .init(
        cluster: "local",
        forceTLS: false,
        enabledTransports: [.ws],
        wsHost: "127.0.0.1",
        wsPort: 6001,
        wssPort: 6001
    )
)
client.connect()
```

```kotlin [Kotlin]
val client = SockudoClient(
    "app-key",
    SockudoOptions(
        cluster = "local",
        forceTls = false,
        enabledTransports = listOf(SockudoTransport.ws),
        wsHost = "127.0.0.1",
        wsPort = 6001,
        wssPort = 6001,
    ),
)
client.connect()
```

```dart [Flutter]
final client = SockudoClient(
  'app-key',
  const SockudoOptions(
    cluster: 'local',
    forceTls: false,
    enabledTransports: <SockudoTransport>[SockudoTransport.ws],
    wsHost: '127.0.0.1',
    wsPort: 6001,
    wssPort: 6001,
  ),
);
client.connect();
```

::

## Disconnect

```js
client.disconnect();
```

## Runtime Requirements

- JavaScript: Node.js >= 22
- Swift: Xcode toolchain with Swift Package Manager
- Kotlin: Java 21+
- Flutter: Flutter 3.35+ and Dart 3.11+
