# Error Codes

Sockudo uses numeric WebSocket close and error codes in the `4000-4399` range.

For protocol v1, the wire event name remains `pusher:error`.
For protocol v2, the server rewrites the prefix to `sockudo:error`.

## Connection Lifecycle

| Code | Meaning | Reconnect |
| ---- | ------- | --------- |
| `4000` | Generic connection or internal error | No |
| `4001` | Application not found | No |
| `4003` | Application disabled | No |
| `4004` | Connection quota exceeded | No |
| `4005` | Path not found | No |
| `4006` | Invalid version string | No |
| `4007` | Unsupported protocol version | No |
| `4008` | Protocol version missing | No |
| `4009` | Authorization or authentication failure | No |

## Capacity And Retry

| Code | Meaning | Reconnect |
| ---- | ------- | --------- |
| `4100` | Capacity exhausted or slow-consumer buffer full | Backoff |
| `4200` | Generic immediate reconnect condition | Immediate |
| `4201` | Pong reply not received | Immediate |
| `4202` | Closed after inactivity timeout | Immediate |

## Channel And Client Event Errors

| Code | Meaning | Reconnect |
| ---- | ------- | --------- |
| `4300` | Channel validation failure | Depends on client action |
| `4301` | Client event rejected or rate limited | Depends on client action |
| `4302` | Watchlist limit exceeded or invalid resume payload | Depends on client action |
| `4303` | Broadcast pipeline failure | Depends on server state |

## Notes

- The canonical mapping lives in [`crates/sockudo-core/src/error.rs`](/Users/radudiaconu/Desktop/Code/Rust/sockudo/crates/sockudo-core/src/error.rs).
- Channel-specific validation errors such as invalid names, namespace permission failures, capability failures, and unauthorized user-limited access are all reported through code `4300` or `4009` depending on whether the failure is authorization-related.
- Protocol v2-only features such as metachannels, wildcard subscriptions, connection metadata, per-namespace client permission flags, and connection capabilities do not affect protocol v1 event names or payload compatibility.
