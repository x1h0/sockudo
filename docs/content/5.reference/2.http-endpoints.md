## Main Server Port (default `6001`)

| Method | Path | Notes |
| --- | --- | --- |
| `POST` | `/apps/{appId}/events` | signed query auth required |
| `POST` | `/apps/{appId}/batch_events` | signed query auth required |
| `GET` | `/apps/{appId}/channels` | signed query auth required |
| `GET` | `/apps/{appId}/channels/{channelName}` | signed query auth required |
| `GET` | `/apps/{appId}/channels/{channelName}/history` | signed query auth required |
| `GET` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations` | list raw annotation events |
| `POST` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations` | publish annotation event |
| `DELETE` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations/{annotationSerial}` | delete annotation event |
| `GET` | `/apps/{appId}/channels/{channelName}/users` | presence only |
| `POST` | `/apps/{appId}/users/{userId}/terminate_connections` | signed query auth required |
| `GET` | `/up` | global health |
| `GET` | `/up/{appId}` | app-specific health |
| `GET` | `/usage` | memory usage stats (disabled via `http_api.usage_enabled: false`) |

Supported `info` tokens:

- `subscription_count`
- `user_count`
- `cache`

## Annotation Endpoints

Annotation endpoints require signed query authentication, `versioned_messages.enabled = true`, global `annotations.enabled = true`, and `annotations_enabled = true` on the resolved channel policy.

| Method | Path | Notes |
| --- | --- | --- |
| `POST` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations` | body fields: `type`, optional `name`, `clientId`, `socketId`, `count`, `data`, `encoding` |
| `DELETE` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations/{annotationSerial}` | optional `socket_id` query param for signed-in capability checks |
| `GET` | `/apps/{appId}/channels/{channelName}/messages/{messageSerial}/annotations` | query fields: optional `type`, `from_serial`, `limit`, `socket_id` |

Publish returns:

```json
{ "annotationSerial": "ann:1" }
```

Delete returns:

```json
{
  "annotationSerial": "ann:2",
  "deletedAnnotationSerial": "ann:1"
}
```

List returns raw `annotation.create` and `annotation.delete` items for audit or repair workflows.

## Server SDK Coverage

| SDK | Channel history helper | Notes |
| --- | --- | --- |
| Ruby | `channel_history` | Already present |
| PHP | `getChannelHistory` | Snake-case alias kept |
| .NET | `FetchHistoryForChannelAsync<T>` | Added in both `PusherServer` and `SockudoServer` |
| Swift | `history(for:options:callback:)` | Typed page model |
| Go | `ChannelHistory` | Typed params and page model |
| Java | `getChannelHistory` | Thin helper over generic signed `get()` |
| Node | `channelHistory` | Thin helper over generic signed `get()` |
| Rust | No dedicated helper | Uses generic signed `get()` by design |
