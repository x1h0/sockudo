# Next.js Core SDK Demo

Core-path quickstart for `@sockudo/ai-transport/react`.

Run from the repository root:

```bash
make demo-core
```

Open `http://127.0.0.1:5175`. The UI uses `TransportProvider`, `useView`, `useCreateView`, `send`,
`edit`, `regenerate`, and a two-pane branch comparison view.

Local-only run against an already running Sockudo server:

```bash
make local-core
```
