# Post-GA Roadmap

## Other Sockudo Client SDKs

- `sockudo-dotnet`: AI Transport bindings over the native realtime client.
- `sockudo-flutter`: AI Transport bindings and demo app.
- `sockudo-kotlin`: Android/JVM bindings with coroutine stream adapters.
- `sockudo-swift`: Swift concurrency bindings and sample app.

## Additional Codecs

- LangChain message codec.
- Provider-specific metadata adapters that preserve opaque codec headers.

## Runtime Validation

- React Native validation through the `@sockudo/client/react-native` subpath.
- Browser compatibility expansion beyond CI smoke coverage.
- Long-running multi-tab soak tests against hosted Sockudo clusters.
